# Robot Adventure
**By:** Landon Nagle

### Instructor Thoughts
This is a really fun and straightforward game. The parkour is hard, but not too hard. The objects and materials chosen look very consistent and provide a bright color palette.

### Tweaks we made post-class
Our latest update fixes the part where the robot will walk off the edge immediately on respawn, making the level a little less frustrating.

### Moving Forward
The game seems pretty much complete, but if you’re looking for things to add to it: Something simple that might look cool is to have a translucent cyan layer over the metallic blue layer to make the water effect a little more realistic, you could also add more things for a longer parkour.
